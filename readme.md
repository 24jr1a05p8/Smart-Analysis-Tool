#### &#x20;                    **Smart Analytics Tool**

#### 

##### **Project Overview**

Smart Analytics Tool is a data analysis application developed using Python. The tool helps users analyze datasets efficiently by providing dataset previews, missing value analysis, statistical summaries, and visualizations.

##### 

##### **Dataset**

This project uses an IPL (Indian Premier League) dataset obtained from Kaggle.

##### 

##### **Features**

* \- CSV File Upload
* \- Dataset Preview
* \- Missing Value Analysis
* \- Statistical Summary
* \- Top 10 Run Scorers Visualization
* \- Top 10 Wicket Takers Visualization
* \- Matches Per Season Analysis
* \- Interactive Data Exploration

##### 

##### **Technologies Used**

* \- Python
* **- Pandas**
* **- Plotly**
* **- Google Colab**

##### 

##### **Project Workflow**

1\. Upload the IPL dataset.

2\. Preview the dataset.

3\. Analyze missing values.

4\. Generate statistical summaries.

5\. Create visualizations.

6\. Extract insights from the data.

##### 

##### **Visualizations**

\- Top 10 IPL Run Scorers

\- Top 10 IPL Wicket Takers

\- Matches Played Per Season

\- Winning Teams Distribution



##### **How to Run**

1\. Open the notebook in Google Colab.

2\. Upload the IPL CSV dataset.

3\. Run all cells sequentially.

4\. View the analysis and visualizations.
